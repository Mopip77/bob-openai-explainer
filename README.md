# bob-openai-explainer

借用 `bob` 的展示能力，使用 `ChatGPT` 快速对所选文本进行解释说明

## Install

在 [Release](https://github.com/Mopip77/bob-openai-explainer/releases) 页面下载并安装

## Usage

建议 `隐藏并钉到语言切换栏`，按需使用

> <img width="500" src="static/example.gif"></img>

## Configuration

| key | defaultValue |
| --- | ----- |
| prompt | 简明扼要地解释 |
